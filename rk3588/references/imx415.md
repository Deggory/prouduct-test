Purpose:

Provide a hardware and integration reference for Sony IMX415 deployments on:

* RK3588
* RK3588S
* Orange Pi 5
* Orange Pi 5 Plus

Target Software:

* openpilot
* sunnypilot
* frogpilot
* OPKR
* KisaPilot

# 1. Overview

Sensor:

Sony IMX415

Type:

CMOS Image Sensor

Interface:

MIPI CSI-2

Common Usage:

* Automotive cameras
* AI vision systems
* Surveillance systems
* Embedded vision

# 2. Why IMX415

Advantages:

* High resolution
* Good low-light performance
* Stable Linux support
* RKISP compatibility
* CSI interface

Suitable for:

Road-facing cameras

NV12

* direct OpenCL processing
* improved performance

# 12. Memory Layout

Possible layouts:

Tight NV12

Padded NV12

Must be detected.

Never assumed.

---

# 13. Tight NV12 Example

1280×720

Expected size:

1382400 bytes

Calculation:

width × height × 1.5

---

# 14. Padded NV12 Example

Stride larger than width.

Expected:

sizeimage larger than:

width × height × 1.5

Must be handled correctly.
