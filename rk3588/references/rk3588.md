Version: 1.0

Purpose:

Provide a hardware-level reference for AI agents integrating:

* OpenPilot

* RK3588
* RK3588S

platforms.

---

# 1. Overview

RK3588 is Rockchip's flagship AI SoC.

Key capabilities:

* 8 CPU cores
* Mali GPU
* Dedicated NPU
* Multiple ISP pipelines
* PCIe
* USB3
* CSI cameras

This makes it suitable for openpilot-style workloads.

---

# 2. CPU Architecture

RK3588 contains:

4x Cortex-A76

4x Cortex-A55

Configuration:

```text
Cluster 0:
4x Cortex-A55

Cluster 1:
4x Cortex-A76
```

---

# 3. CPU Performance

Typical frequencies:

2.2–2.4 GHz
